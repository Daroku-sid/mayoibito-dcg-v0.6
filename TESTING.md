# v0.4 実機テスト チェックリスト

配布前の最終確認です。**PC Chrome** と **スマートフォン実機**（できれば iPhone Safari と Android Chrome の両方）で見てください。

すべてを一度にやる必要はありません。**A と B だけでも通れば、知り合いに渡せます。**

---

## A. まず通す（これが通らなければ配布しない）

- [ ] スタート画面が出て、タップするとモード選択へ進む
- [ ] モード選択に「対戦」「カード」「遊び方」と、左上に「設定」がある
- [ ] 対戦 → CPU対戦 → 設定 → 開始、までたどれる
- [ ] CPU対戦を最初から最後まで遊べる（難易度：弱）
- [ ] ひとり回しを最初から最後まで遊べる
- [ ] CPU観戦が最後まで自動で流れる
- [ ] リザルトが出て、「モード選択へ戻る」で戻れる
- [ ] 画像がすべて表示される（枠だけの空白カードが無い）

---

## B. スマートフォンでの操作

いちばん壊れやすいところです。

- [ ] 手札の束をタップすると扇に広がる
- [ ] 扇の上を**横になぞる**とカードが切り替わる
- [ ] カードを**斜め上に持ち上げて**盤面へ置ける（なぞりに取られない）
- [ ] 置ける場所が光る
- [ ] 怪異を相手の人間へドラッグすると矢印が出る
- [ ] 「追跡を確定」でターンが終わる
- [ ] カードをタップで簡易表示、長押しで拡大表示
- [ ] 素早く2回タップしても**画面が拡大されない**
- [ ] 横向きにすると注意が出て、縦に戻すと消える
- [ ] ボタンが押しやすい大きさか（押しにくいものがあればメモ）

---

## C. CPU対戦

- [ ] CPUの手札が裏向きのまま。タップも長押しもできない
- [ ] CPUが使うカードが中央に一時公開される
- [ ] 「CPU思考中…」が出る
- [ ] CPUのターン中は操作できない
- [ ] **CPUのターン中に空白をタップすると手札が閉じる**
- [ ] 自分の陣営が常に画面下にある（上下が入れ替わらない）
- [ ] メインフェイズ開始時に「あなたのメインフェイズ」が出る
- [ ] 出せるカードだけがうっすら光る。CPUのターンには光らない
- [ ] 難易度「弱」に勝てる／「理不尽」が明らかに強い
- [ ] ログを開いても**CPUの手札が分からない**（「初期ドロー：CPU 5枚」と出る）

---

## D. ミラー対戦

- [ ] CPU対戦で自分もCPUも「ヨマモリ村」を選んで始められる
- [ ] 両側とも同じ初期人間で始まる
- [ ] ひとり回しのミラーで、呼び名が「プレイヤー1／プレイヤー2」になる

---

## E. 観戦

- [ ] 一時停止が効く。**演出の途中で固まらない**
- [ ] 一時停止と再開を連打しても壊れない
- [ ] **一時停止 → 設定を開く → 閉じる → 止まったまま**
- [ ] 速度3段階が切り替わる
- [ ] 「観戦を終了」で確認が出て、観戦設定へ戻れる

---

## F. メニューと設定

- [ ] 歯車ボタンでメニューが開く
- [ ] メニューから遊び方を開いて、閉じると同じ対戦へ戻る
- [ ] **CPUのターン中に設定を開くと、閉じるまでCPUが進まない**
- [ ] リタイアで確認が出て、モード選択へ戻れる
- [ ] 効果音が鳴る。ON/OFF が効く。音量スライダーが効く
- [ ] **最初のタップまで音が鳴らない**（これが正しい挙動）
- [ ] 設定を変えて**ブラウザを再読み込みしても残っている**
- [ ] 「設定を初期化」で戻る

---

## G. リザルトと報告

- [ ] 対戦の条件（デッキ・難易度・先攻・シード）が正しい
- [ ] 「対戦結果をコピー」でクリップボードに入る
- [ ] コピーできない場合、下に文章の枠が出る
- [ ] 「同じ設定で再戦」でランダム項目が選び直される
  （CPUのデッキを「ランダム」にすると分かりやすい）
- [ ] リザルトが縦に長すぎない

## G2. デッキ作りと自作デッキでの対戦（v0.4で追加）

### カード一覧
- [ ] 6列で並び、右下に所持枚数（×4）が出る
- [ ] **フィールドは区切りの下に、横向き4列**で出る
- [ ] 検索・特徴検索・種類・コスト・並び替えが効く
- [ ] 絞り込んでも**上下のバーが動かない**

### デッキ一覧・確認
- [ ] 公式2つが出て、**編集と削除のボタンが出ない**
- [ ] 自作デッキでは編集・削除が出て、**削除の確認が出て実際に消える**
- [ ] 40枚が8×5で並び、足りないぶんは空き枠になる
- [ ] **最終更新日時**が出る（公式には出ない）
- [ ] 画像保存でPNGが落ちてくる

### デッキ編成
- [ ] カードのドラッグが**指に付いてくる**（もたつかない）
- [ ] 帯の**横スワイプはスクロール**、**上へのドラッグは追加**
- [ ] 40枠から帯へ引くと削除できる
- [ ] **長押しでエースカード**が設定でき、光って見える
- [ ] フィールドをタップして選び直すと、**主人公が入れかわり他は残る**
- [ ] 40枚で41枚目が止まり、理由が出る
- [ ] **主人公は外せない**（「主」の印が付く）
- [ ] 保存して戻り、また開くと内容が残っている

### 自作デッキで対戦
- [ ] CPU対戦とひとり回しで、デッキ選択が**カード型**になっている
- [ ] 自分側では**自作デッキが選べる**
- [ ] **CPU側には自作デッキが出ない**（公式2つとランダムだけ）
- [ ] 40枚に満たないデッキは**暗くて選べず、押すと理由が出る**
- [ ] 自作デッキで**最後まで対戦できる**
- [ ] 対戦画面のカード画像が正しく出る

不具合を見つけたら、これだけあれば追えます。

```
端末：iPhone 14 / Safari
何をした：CPU対戦（強・自分が村）で、3ターン目にコハクを出した直後
どうなった：画面が止まって操作できない
シード：MAYO-1A2B-3C4D
```

シードはリザルトの「対戦結果をコピー」か、対戦中のログの1行目にあります。
